Automated Water Overflow Control

Intermediate Project
MAAN English Public School
Arduino IDE C/C++ Ultrasonic Sensor Relay Modules

Project Overview

In many residential and educational complexes in water-stressed regions, manual water pump switching is a common source of resource waste. Forgetting to turn off a high-voltage pump leads to massive water overflow and excessive electrical consumption.

The Automated Water Overflow Control System was developed during my intermediate studies at MAAN English Public School to address this challenge. It implements an autonomous closed-loop feedback controller that continuously checks the tank's water level and toggles a high-power AC pump via a safe low-voltage relay trigger, preventing any overflow.

Closed-Loop Feedback Control Design

The system works as a hysteresis-based closed-loop controller:

  • Ultrasonic Ranging: Deployed the HC-SR04 ultrasonic distance sensor. The microcontroller triggers a 10µs pulse on the transmitter pin, causing it to emit an 8-cycle ultrasonic burst. It then measures the echo pin pulse duration to compute the exact air gap distance above the water surface.
  • Hysteresis Thresholds: To prevent rapid, unwanted relay switching ("chattering") due to surface ripples, a double-threshold system was programmed:
    • Low Water Limit (Empty): Triggers the relay ON when the distance exceeds the maximum threshold (e.g. 80cm from the sensor).
    • High Water Limit (Full): Triggers the relay OFF when the distance drops below the minimum safety threshold (e.g. 15cm from the sensor).
  • Optocoupler Isolation: Used a relay driver board with optocoupler isolation, isolating the sensitive microcontroller circuit from high-voltage AC spikes generated by the pump.

Firmware Design

The code below demonstrates the pulse duration calculation and the output relay trigger system:

water_pump_control.ino C++ (Arduino) 
const int trigPin = 9;
const int echoPin = 10;
const int relayPin = 8;

// Hysteresis calibration in centimeters
const int DIST_EMPTY = 80; // Distance to bottom (Pump starts)
const int DIST_FULL  = 15; // Distance to top safety limit (Pump stops)

void setup() {
  Serial.begin(9600);
  pinMode(trigPin, OUTPUT);
  pinMode(echoPin, INPUT);
  pinMode(relayPin, OUTPUT);
  
  // Set default state (Relay active-low or active-high)
  digitalWrite(relayPin, HIGH); // Assuming active-low: Relay OFF
}

void loop() {
  long duration;
  int distance;
  
  // Trigger sensor pulse
  digitalWrite(trigPin, LOW);
  delayMicroseconds(2);
  digitalWrite(trigPin, HIGH);
  delayMicroseconds(10);
  digitalWrite(trigPin, LOW);
  
  // Measure pulse duration on echo pin (in microseconds)
  duration = pulseIn(echoPin, HIGH);
  
  // Convert duration to distance (Speed of sound = 340m/s -> 29.1 us/cm)
  distance = duration / 29.1 / 2;
  
  if (distance > 0 && distance < 200) { // Filter out-of-range sensor noise
    Serial.print("Water distance to sensor: ");
    Serial.print(distance);
    Serial.println(" cm");
    
    // Hysteresis control loop
    if (distance >= DIST_EMPTY) {
      digitalWrite(relayPin, LOW);  // Turn ON Pump (Relay ACTIVE)
      Serial.println("Tank empty. Pump ON.");
    } 
    else if (distance <= DIST_FULL) {
      digitalWrite(relayPin, HIGH); // Turn OFF Pump (Relay INACTIVE)
      Serial.println("Tank full. Pump OFF.");
    }
  }
  
  delay(1000); // 1-second sample loop interval
}

Project Details

  • Sensors: HC-SR04 Ultrasonic Distance Sensor
  • Relay Controller: 5V single-channel relay module
  • Firmware Base: Arduino C/C++
  • Target Pump Load: 220V AC / 1HP Pump
  • Calibration Style: Dual-threshold Hysteresis Loop

Key Highlights

  • Low-cost micro-controller prototyping.
  • Optocoupled relay interface for high-voltage protection.
  • Robust distance range filtering to prevent sensor misfires.
  • Hysteresis triggers to extend the life of the relay contacts.
